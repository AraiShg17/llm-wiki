# Project State

このファイルは、AIが作業開始時に現在の状態を短く把握するための一時的な情報です。

恒久的な仕様はここに書きません。仕様は `features/`、構成は `architecture.md`、作業方法は `skills/` を参照してください。

## 更新ルール

- 作業開始前に読む
- 作業完了時に必要なら更新する
- 古い情報を残さない
- 判断に使えない詳細ログを書かない
- PR、未完了作業、既知の負債だけを短く残す
